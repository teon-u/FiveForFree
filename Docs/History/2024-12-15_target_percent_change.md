# TARGET_PERCENT 변경: 5% → 1%

**변경일**: 2024-12-15
**변경 항목**: `TARGET_PERCENT` 설정값
**변경 전**: 5.0%
**변경 후**: 1.0%

---

## 변경 이유

### 1. 클래스 불균형 문제 완화

기존 5% 임계값의 문제점:
- 5% 이상 상승/하락은 전체 데이터의 약 3-5%만 해당
- 극심한 클래스 불균형으로 인해 모델 학습이 어려움
- Positive 샘플이 너무 적어 패턴 학습에 한계

1% 임계값 적용 후:
- Positive 샘플 비율 증가로 균형 잡힌 학습 가능
- 모델이 더 다양한 패턴을 학습할 수 있음

### 2. 현실적인 단기 트레이딩 목표

- 60분 내 5% 변동은 매우 드문 이벤트
- 1% 변동은 고변동성 종목에서 현실적으로 달성 가능한 목표
- 더 자주 발생하는 이벤트를 예측하므로 실용성 증가

### 3. 데이터 활용도 향상

- 1% 변동 이벤트가 더 자주 발생하므로 학습 데이터 양 증가
- 백테스팅 시 더 많은 거래 기회 포착 가능
- 모델 성능 평가의 통계적 유의성 향상

### 4. 리스크 관리

- 작은 목표 수익률로 더 빈번한 이익 실현 가능
- 손실 제한과 함께 사용 시 리스크/리워드 비율 개선
- 복리 효과를 통한 장기 수익률 향상 기대
