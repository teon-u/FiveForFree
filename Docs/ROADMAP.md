## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **목표** | NASDAQ 주식의 1시간 내 1% 이상 가격 변동 예측 |
| **대상** | 고변동성 NASDAQ 종목 (거래량/상승률 Top 100) |
| **모델** | XGBoost, LightGBM, LSTM, Transformer, Ensemble |
| **데이터** | Yahoo Finance + Finnhub (무료) |

## 🔥 현재 진행 중: 실전 투자 효용가치 점검

> **목표**: 12/25 시스템 정상 동작 → 12/26 실전 투자 테스트

### Phase 4: 실전 검증 (2025-12-24 ~ 12-26)

| 단계 | 작업 | 일정 | 상태 |
|------|------|------|------|
| 1 | 시스템 전체 기동 테스트 | 12/24 | ⬜ 대기 |
| 2 | 데이터 수집 및 모델 재학습 | 12/24 | ⬜ 대기 |
| 3 | API/UI 정상 동작 확인 | 12/25 | ⬜ 대기 |
| 4 | 실시간 예측 모니터링 | 12/26 | ⬜ 대기 |
| 5 | Paper Trading 기록 | 12/26~ | ⬜ 대기 |
| 6 | 성과 분석 및 Go/No-Go 결정 | 12/27 | ⬜ 대기 |

### 투자 판단 기준

| 지표 | 목표 | Go 조건 |
|------|------|---------|
| Precision (상승 예측 정확도) | 60%+ | ✅ 실전 가능 |
| Win Rate (수익 거래 비율) | 55%+ | ✅ 실전 가능 |
| Signal Rate (신호 발생률) | 10건/일+ | 충분한 기회 |

### 관련 명령어
```bash
python scripts/check_data.py        # 데이터 상태 확인
python scripts/train_all_models.py  # 모델 학습
python scripts/run_system.py        # 시스템 실행
```
